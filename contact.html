Contact

There are many ways to contact us. If you'd like to leave a nice message for the developers, you can do so by using the comments section of the BukkitDev page. However, if you have an issue with the plugin or website, or want to suggest a feature, please use the appropriate method listed below:


Want to help out

Feel free to submit a Pull Request on our GitHub repository. You could make a pull request of improvements or just knock out a few bugs from our GitHub issue tracker. Donations also help out so feel free to click the donation button at the bottom right of the page

Want to report a bug?

We use the GitHub issue tracker to easily manage bug reports and plugin issues. Simply visit the issue tracker, read the issue requirements, and then create an issue detailing your problem with the plugin. Be sure to supply all the requested information and be ready to give more if asked. One of the developers will take a look at it soon and try to help resolve the problem.

Want to suggest a feature?

We love feature suggestions! Simply hop on to #harrypotterspells (irc.esper.net) and talk to one of the developers there. If no one is present, simply leave a message stating exactly what you'd like to see in the plugin. We'll add it to the Trello board and you can vote for the feature to be added.

Want to report an issue on the site?

The website is still a work in progress. If you have an issue with the site or find a hole in the documentation, please hop on #harrypotterspells (irc.esper.net), mention "np98765", and state the issue that you found. We're still working on the website and we appreciate any feedback you may have!
